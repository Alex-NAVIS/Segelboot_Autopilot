# Segelboot_Autopilot


Segelboot Autopilot (ESP32)Ein Open-Source-Autopilot für Segelboote, basierend auf dem ESP32-Mikrocontroller. Das Projekt kombiniert präzise Kursberechnungen in C/C++ mit einer komfortablen, plattformunabhängigen Steuerung über ein integriertes HTML-Webinterface.🚀 FeaturesESP32-Basis: Nutzt die Rechenleistung und die integrierten WLAN/Bluetooth-Funktionen des ESP32.Webinterface (HTML/JS): Kabellose Steuerung und Konfiguration direkt über das Smartphone, Tablet oder den Laptop.Kursbarmodi: Unterstürzt Kompasskurs (Heading Hold) und optional Windmodus (Apparent Wind Hold).PID-Regelung: Implementiert einen anpassbaren PID-Regelkreis für eine stabile und sanfte Ruderbewegung.🛠️ Hardware-Komponenten (Beispielhaft)Um diesen Autopiloten aufzubauen, werden typischerweise folgende Komponenten benötigt:Mikrocontroller: ESP32 Development BoardSensorik: IMU (z. B. MPU9250 oder BNO055) für Kompass und NeigungsausgleichAktuator: Linearantrieb, Scheibenwischermotor oder hydraulische Pumpe mit passender H-Brücke (z. B. BTS7960)Feedback (Optional): Ruderlagenrückgeber (Potentiometer)📁 Projektstrukturtext├── ESP32_Autopilot/      # Quellcode für den ESP32 (C++/C)
│   ├── src/              # Logik, PID-Regler und Sensor-Auswertung
│   └── data/             # Webserver-Dateien (HTML, CSS, JS)
└── README.md             # Diese Dokumentation
